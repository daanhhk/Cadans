# Screenshots — Cadans (visuele richtlijn)

Verse renders van de schermen, op 390px mobiel-breed. **Richtlijn, geen pixel-bron** —
de exacte waarheid zit in `../src/` (HTML/JSX) en `../docs/`. Gebruik deze om de eindstaat
in één oogopslag te zien zonder de code te draaien.

| Bestand | Toont |
| --- | --- |

| `instellingen - jouw coach.png` | Instellingen → "Jouw coach": naamveld + suggestie-chips + "Cadans blijft de app-naam"-noot |
| `schema-vandaag - coach-gereedheidsbanner.png` | **Verbeterpunt 1** — de Vorm→Schema-brug: gereedheid 54 → VO2max verschoven, vandaag rustige Z2 |
| `niveau-tab - deel 1 -snapshot - progressie-.png` | Niveau: vermogen-snapshot (W/kg benadrukt) + progressie-grafiek (switch W/kg · Fitheid) |
| `niveau-tab - deel 2 -rijdersprofiel - doel-gap-.png` | Niveau: rijdersprofiel (power-curve, Fase 2) + doel-gap vs Girona |

**Let op (bekend, = verbeterpunt 3):** in `niveau-tab - leeg.png` toont het Rijdersprofiel nog
de volledige power-curve i.p.v. een lege staat. Dat is precies het nog-te-bouwen punt 3 in
`../README.md` — de screenshot toont bewust de huidige staat.

De Niveau-tab is opgeknipt in 3 delen (deel 1–3) omdat het scherm langer is dan één viewport;
samen vormen ze het volledige tabblad van boven naar onder.
